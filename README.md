# -nskeskyen
Projekt 2. Semester - Ønskeskyen
Ønskeskyen er en webapplikation udviklet i Java med Spring Boot, hvor brugere kan oprette og dele ønskelister.

Systemet gør det muligt for brugere at:

Oprette en konto og logge ind
Oprette og administrere ønskelister
Tilføje, redigere og slette ønsker
Dele ønskelister via et offentligt link
Reservere ønsker på andres ønskelister

Formålet med systemet er at gøre det nemt at koordinere gaveønsker, så flere personer ikke køber den samme gave.

Applikationen anvender:

Spring Boot (backend)
Spring Security (login og sikkerhed)
Spring Data JPA (database)
Thymeleaf (frontend)

Systemet har fokus på:

Brugervenlighed
Sikkerhed (login, password hashing)
Simpel social funktionalitet
